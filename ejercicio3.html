Explicaciones HTML

Parrafos

para utilizar el elemento parrafo debemos escribir el contenido dentro de la etiqueta <p> y la etiqueta </p>

Anchors

las etiquetas <a> en html tienen que llevar el argumento href="" en el que tenemos que poner la URL de la imagen a la que queremos enlazar. Tambien habra que añadirle un atributo alt="" que permita poner un texto adicional para el lector de pantalla en caso de fallo en el link.

Tambien podemos incluir el atributo title="" que permite añadir un titulo que se muestre al sobreponer el raton sobre la imagen. en este caso quiero poner:

"Este es el tittle que explica que el title de este link dice title"
